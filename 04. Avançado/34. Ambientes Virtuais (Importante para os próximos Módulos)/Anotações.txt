AULA_01: O que é um Ambiente Virtual

    Usar ferramentas que funcionam apenas com versões específicas do Python
    Isolar de todo o resto tudo aquilo que é estritamente necessário para o programa rodar. 
    Importante para sites e programas que vão ser distribuídos

AULA_03: Criando, Ativando e Removendo Ambientes Virtuais

    Documentação: https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-environments.html

    Criar Ambiente virtual com versão especifica:
        conda create -n meuambientevirtual python=3.6
    
    Remover ambiente virtual:
        conda remove --name meuambientevirtual --all

    Ativar ambiente:
        conda activate meuambientevirtual

    Desativar ambiente:
        conda deactivate


AULA_04: Observações Importantes de Ambientes Virtuais

    Quando você cria um ambiente virtual, ele vem praticamente sem nada, 
    então você precisa usar o pip ou o conda para instalar cada pacote que for usar
