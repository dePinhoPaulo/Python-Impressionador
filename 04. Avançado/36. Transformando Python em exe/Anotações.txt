AULA_2: Instruções Importantes

    Seu código deve estar funcionando sem erros no jupyter
    Transformar o código jupyter em scripts python padrão (extensão .py). Seu código deve estar funcionando nesse formato também.
    Usar uma biblioteca de conversão (pyinstaller ou auto-py-to-exe) para transformar o código em executável.
    Testar e adaptar o que for necessário.


AULA_3: Python para exe com Arquivos Simples

    Biblioteca pyinstaller

    No cmd, executar o comando de conversão para exe:
        pyinstaller -w nome_do_programa.py


AULA_4: Diminuindo Tamanho de Arquivos Executáveis

    Para diminuir o tamanho do arquivo, criar um ambiente virtual para garantir que teremos apenas as bibliotecas importantes.

    Criar o ambiente virtual
    Executar o nosso código por dentro do ambiente virtual
    Identificar erros e instalar bibliotecas que faltam, apenas as que o programa pede.
    Instalar o pyinstaller e transformar em executável o programa Python


AULA_5: Adaptando o código de Arquivos Complexos

    Codigo de automação Selenium em EXE:
        auto-py-to-exe para transformar o arquivo python em executável
        pathlib ou os para adaptar todos os "caminhos dos arquivos"
        Alternativamente, podemos usar o tkinter para permitir a gente escolher manualmente o arquivo, independente do computador que vamos rodar o programa
