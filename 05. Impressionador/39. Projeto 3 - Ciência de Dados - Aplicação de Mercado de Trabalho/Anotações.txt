AULA_17: Criando uma função para os limites superior e inferior

    Função para Análise de Outliers:
        Regra = valores abaixo de Q1 - 1.5xAmplitude e valores acima de Q3 + 1.5x Amplitude. Amplitude = Q3 - Q1

        def limites(coluna):
            q1 = coluna.quantile(0.25)
            q3 = coluna.quantile(0.75)
            amplitude = q3 - q1
            return q1 - 1.5 * amplitude, q3 + 1.5 * amplitude
