AULA_03: Como decidir por qual caminho automatizar tarefas?

    Recomendações das automações:
        RPA - Pyautogui
            + Simples
            + funciona em qualquer programa
            + se passa por voce
            - lento
            - fragil
            - naõ funciona em 2° plano
        
        Bibliotecas/Códigod/APIS
            + Melhor solução
            + robusto
            + seguro
            + roda em 2° plano
            + escalavel
            - mais dificil tecnicamente
            - pode ter custos

        Selenium/Playwright
            + roda em 2° plano
            + escalavel, mas não tanto
            + mais robusto que RPA, mas menos que API
            - só funciona no navegador
